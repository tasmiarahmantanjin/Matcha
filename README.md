
# Matcha

Instagram like web app with PERN stack (PostgreSQL, Express, React, Node)


## Run Locally

Clone the project

```bash
  git clone https://github.com/tasmiarahmantanjin/Matcha.git
```

Go to the project directory

```bash
  cd Matcha
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  cd server
  npm run dev
```
Start the client

```bash
  cd client
  npm run start
```
